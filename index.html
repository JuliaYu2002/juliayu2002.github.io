üöÄ Featured Projects

Sweep & Set Independent Study Project

Using Unity and an Oculus Quest 3, I created an AR experience that can interface with ROS controlled Dynamixel motors. The user can make sweeping gestures with their right hand and select motors with their left hand. The motors will move in the direction of the gesture and stop when the user releases the gesture. I worked under Professor Harald Haraldsson in Cornell Tech's XR Collaboratory.

Tech: Oculus Quest 3, Unity, ROS2, Dynamixel motors, C#

Gravity Sketch Class Project Implementation

With a group of 3 other students, we created a recreation of the application Gravity Sketch for the Meta Quest. Includes features like snap back, color changing, and more.

Tech: Unity, Oculus Quest 2, Gravity Sketch application

Hexapod Teleoperation Revamp Independent Study Project

I examined a PhantomX Hexapod Mk-III robot to see if it was still operable. This used 18 daisy-chained Dyanmixel motors to move 6 legs. I created a Python script to move its legs to user-specified positions by adapting the Dynamixel SDK. I worked under Professor Kevin Huang in Smith College's Scram Lab.

Tech: Python, Dynamixel SDK, PhantomX Hexapod Mk-III

Bootleg Pokemon Game

My final project for my computer graphics class. I utilized Blender to make a player model and to orient models of Pokemon obtained online, ThreeJS for creating the actual scene and JavaScript for getting the game intricacies to work.

Tech: Blender, ThreeJS, JavaScript
